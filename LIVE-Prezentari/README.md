## Prezentare HTML

### Cum deschizi
```bash
# Windows
start prezentare-vanzare.html

# Mac/Linux
open prezentare-vanzare.html
```

### Navigare

| Acțiune | Control |
|---------|---------|
| Slide următor | `→` / `Space` / Click dreapta |
| Slide anterior | `←` / Click stânga |
| Fullscreen | Tasta `F` sau buton |
| Prima/Ultima | `Home` / `End` |
| Touch | Swipe stânga/dreapta |

### Design

- **Background:** `#E8F0EC` (verde mint)
- **Accent:** `#1A4A4A` → `#2D7A7A` (gradient teal)
- **Text:** `#1A3A3A` (verde închis)
- **Font:** Inter (Google Fonts)
- **Aspect ratio:** 16:9 (1920x1080)

## Principii aplicate (Hristosenko)

- **Privivka** — Anunț la început că la final e ceva special
- **Podul** — Întrebări care conectează problema la soluție
- **Agitația** — "Singură nu poți" / mediul te trage înapoi
- **4 Nivele Transformare** — A avea → A face → A simți → A fi
- **Exclusivitate** — Doar lista de preînregistrare
- **CTA multiplu** — 7+ apeluri cu emoții diferite
- **Valoare 10x Preț** — Focus pe transformare, nu lecții
